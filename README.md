# My Time Logger

時間記録アプリ - 何にどれだけの時間を使ったかを記録・可視化するPWAアプリ

## 技術スタック

- **フロントエンド**: Vite + React 18 + TypeScript
- **UI**: TailwindCSS + shadcn/ui
- **状態管理**: TanStack Query + Zustand
- **ルーティング**: React Router
- **DB**: Supabase (PostgreSQL)
- **ホスティング**: Vercel

## 開発

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```
